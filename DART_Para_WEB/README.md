# SchoolOfNet_DART_PARA_WEB
Curso do schoolofnet Dart para Web


Será necessário instalar algumas perramentas, para isso vamo usar o pub que já vem instalado junto com o DAT

pub --help

Instalando pacotes globalmente

stagehand:

Paco que gera um stage de trabalho

pub global activate stagehand

stagehand --help

OBS: se o comando acima não funcionar, tem que colocar o seguinte caminho nas variaveis de ambiente:

C:\Users\carlos.marian\AppData\Roaming\Pub\Cache\bin

Para criar uma aplicação web simples:


stagehand web-simple

Carrega as dependencias:

pub get

Para rodar a aplicação:

pub global activate webdev 

Para executar:

webdev serve
